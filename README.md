# AURA — Premium Audio Experience

Welcome to the **AURA** repository. This project is a front-end e-commerce web application for a luxury audio brand. It showcases premium wireless headphones, earbuds, and speakers, featuring a sleek, modern, and minimal aesthetic designed to provide an immersive shopping experience.

## 🌟 Project Description

AURA is engineered to feel like a high-end, Apple-inspired luxury audio store. The project leverages clean HTML, custom CSS, and vanilla JavaScript to deliver smooth scroll-triggered animations, interactive galleries, and responsive bento-box layouts without relying on heavy frameworks. 

**Key Features:**
- **Premium UI/UX:** A minimalist, bright aesthetic with glassmorphism touches and carefully tuned spacing.
- **Cinematic Showcases:** Integrated video backgrounds and fluid reveal-on-scroll animations.
- **Interactive E-Commerce Elements:** Custom product sliders, interactive "Like" buttons, dynamic color swatches, and cart functionality.
- **Responsive Design:** Fully fluid layouts utilizing CSS Grid and Flexbox that scale perfectly from large desktop monitors down to mobile devices.

## 🗂️ File Structure

The repository maintains a flat and straightforward structure for ease of development:

```text
aura/
├── index.html          # The main homepage/landing page (Hero, New Arrivals, Trending)
├── product.html        # Detailed product view (Gallery, Specs, Bento Grid, Reviews)
├── README.md           # Project documentation
└── assets/             # Directory for static resources
    ├── style.css       # Core stylesheet (Design tokens, utility classes, and components)
    ├── script.js       # Core interactivity (Sliders, observers, UI toggles)
    └── (media files)   # Images, icons, and cinematic video assets (.jpg, .png, .mp4)
```

## 🖋️ Typography & Fonts

AURA uses a carefully curated typography stack to maintain its tech-luxury feel. The fonts are imported directly via Google Fonts.

*   **[Inter](https://fonts.google.com/specimen/Inter)** (Weights: 300, 400, 500, 600, 700, 800, 900)
    *   Used for the primary body text, UI elements, and technical specifications to ensure maximum legibility and a clean, modern look.
*   **[Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk)** (Weights: 500, 600, 700)
    *   Used for accent typography and specific stylized headers to add a subtle, futuristic technological edge to the brand.

## 🚀 Getting Started

Since this is a static website, no build tools or package managers are required. 

1. Clone the repository.
2. Open `index.html` in your preferred modern web browser.
3. For the best development experience, consider using a local development server (like VS Code's "Live Server" extension) to properly test the scroll animations and video autoplay functionalities.

## 🛠️ Technologies Used

*   **HTML5:** Semantic markup for structured content.
*   **CSS3:** Vanilla CSS utilizing Custom Properties (Variables), Grid, Flexbox, clamp() functions, and media queries for fluid responsiveness.
*   **JavaScript (ES6):** Vanilla JS for DOM manipulation, Intersection Observers (scroll reveals), and component interactivity.
